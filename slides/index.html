First things first

  1. Go to GitHub.com
  2. Create a GitHub account
  3. Create a new repository named username.github.io
  4. Check your email for GitHub's verifaction request

The
in GitHub

  • git is a distributed version control system
  • created in 2005 by Linus Torvalds, the namesake of the free operating system commonly known as Linux

Distributed version control system?

Distributed means...

everyone working with a project in git has a copy of the full history of the project, not just the current state of the files.

Distributed version control system?

Version control system means...

software designed to keep track of the changes made to files over time.

Version control systems you've already used

  • The keyboard combo ctrl+z (undo)
  • MS Word
  • Google Docs
  • and more...

Distributed version control systems you've already used

  • Sharing MS Word files with others with "Track changes" enabled
  • Sharing Google Docs with others
  • Wikipedia page histories
  • and more...

GitHub

GitHub is...

  • A website where you can upload and work on git repositories
  • A social network and collaboration space to work with others with all of git's powers
  • A place to host free websites (like these slides)
  • A major place to get open source software and data like...

The variety of data on GitHub

  1. Web design frameworks like Bootstrap & Basscss
  2. R packages
  3. Data behind NY Times analysis
  4. Obama's 2016 budget
  5. Lots more!

Social on GitHub

  • Follow people
  • Favorite individual repositories
  • Wikis
  • Create issues and pull requests
  • Ask for (and receive!) help
  • Give feedback
  • Fork the hardwork of others
